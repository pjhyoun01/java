	int 판매 - 거스름돈

백화점 Store
	int 판매 - 거스름돈
	int 포인트 사용 - 사용할 포인트
		10_000 포인트 이상일 경우만 사용 가능
		원하는 금액 만큼 사용 가능
		사용한 포인트 만큼 보유 포인트 감소
	int 포인트 적립 - 적립될 포인트
		등급에 따라 차등 포인트 적립
		일반		0.5%
		VIP		x
		VVIP	3%
	int 할인 - 할인될 금액
		일반		x
		VIP		3%
		VVIP	10%
